# Extract useEditorInput hook

Extract keyboard commands and input transforms from USFMPlugin.tsx into src/app/domain/editor/hooks/useEditorInput.ts.

Logic to extract:
- redirectParaInsertionToLineBreak register (USFMPlugin.tsx lines 193-194)
- KEY_DOWN_COMMAND listener (lockImutableMarkersOnType) (USFMPlugin.tsx lines 207-217)
- KEY_DOWN_COMMAND listener (moveToAdjacentNodesWhenSeemsAppropriate) (USFMPlugin.tsx lines 218-228)
- PASTE_COMMAND (lockImmutableMarkersOnPaste) (USFMPlugin.tsx line 229)
- CUT_COMMAND (lockImmutableMarkersOnCut) (USFMPlugin.tsx line 230)
- Node Transforms: textNodeTransform, inverseTextNodeTransform (USFMPlugin.tsx lines 176-190)

The hook must:
1. Accept editor: LexicalEditor as argument
2. Use useWorkspaceContext internally for:
   - mode
   - appSettings.markersMutableState
   - appSettings.markersViewState
   - projectLanguageDirection
3. Register USFMTextNode transform in useEffect:
   - textNodeTransform({ node, editor, editorMode: mode, markersMutableState, markersViewState, languageDirection: projectLanguageDirection })

4. Register redirectParaInsertionToLineBreak(editor)

6. Register KEY_DOWN_COMMAND with moveToAdjacentNodesWhenSeemsAppropriate(editor, event)
   - Priority: COMMAND_PRIORITY_HIGH
7. Register PASTE_COMMAND with lockImmutableMarkersOnPaste(editor)
8. Register CUT_COMMAND with lockImmutableMarkersOnCut(editor)
9. Return cleanup function that calls all unregister functions
10. Export function useEditorInput(editor: LexicalEditor)
