# Extract useEditorView hook

Extract visual-only update logic from USFMPlugin.tsx into src/app/domain/editor/hooks/useEditorView.ts.

Logic to extract:
- markersInPreview ref (USFMPlugin.tsx line 50)
- editor.registerUpdateListener for wysiPreview/toggleShowOnToggleableNodes (USFMPlugin.tsx lines 153-167)
- syncRefScrollUnregister (syncReferencePaneSid) (USFMPlugin.tsx lines 232-235)
- cursorCorrectionUnregister (correctCursorIfNeeded) (USFMPlugin.tsx lines 238-252)

The hook must:
1. Accept editor: LexicalEditor as argument
2. Use useWorkspaceContext internally for:
   - appSettings.markersViewState
   - mode
   - referenceProject?.referenceProjectId
3. Maintain internal markersInPreview ref: useRef(new Set<NodeKey>())
4. Register wysiPreview update listener in useEffect:
   - if (markersViewState !== EditorMarkersViewStates.WHEN_EDITING) return
   - toggleShowOnToggleableNodes({ editor, editorState, markersViewState, currentActive: markersInPreview.current, markersMutableState, setCurrentActive: (activeNodes) => { markersInPreview.current = activeNodes } })
5. Register syncReferencePaneSid(editor, referenceProject?.referenceProjectId)
6. Register cursorCorrectionUnregister update listener:
   - if (mode !== EditorModes.WYSIWYG) return
   - if (tags.has(EDITOR_TAGS_USED.programmaticDoRunChanges)) return
   - if (editorState.isEmpty()) return
   - correctCursorIfNeeded(editor)
7. Return cleanup function that calls all unregister functions
8. Export function useEditorView(editor: LexicalEditor)

Reference files:
- src/app/domain/editor/plugins/USFMPlugin.tsx
- src/app/domain/editor/listeners/livePreviewToggleableNodes.ts
- src/app/domain/editor/listeners/syncReferencePaneSid.ts
- src/app/domain/editor/utils/cursorCorrection.ts
- src/app/data/editor.ts (for EditorMarkersViewStates, EditorModes, EDITOR_TAGS_USED)

## Design

markersInPreview ref must be maintained internally by the hook.
All three listeners (wysiPreview, syncRef, cursorCorrection) must be registered inside useEffect.
Cleanup must call: wysiPreview, syncRefScrollUnregister, cursorCorrectionUnregister.

## Acceptance Criteria

useEditorView hook created and exported. Hook maintains internal markersInPreview ref, registers wysiPreview listener with markersViewState check, registers syncReferencePaneSid, registers cursorCorrection with mode/tag/empty checks, and returns proper cleanup.
