# Register cursor correction listener in USFMPlugin.tsx useEffect

Register the cursor correction update listener in the USFMPlugin.tsx useEffect hook. This listener should run on every editor update but early exit when not in Regular (WYSIWYG) mode or when update was triggered programmatically.

File to modify: src/app/domain/editor/plugins/USFMPlugin.tsx

In the existing useEffect hook (around line 155-202 in the plan), add:

```typescript
useEffect(() => {
    if (mode === EditorModes.SOURCE) {
        console.log("mode === EditorModes.SOURCE");
        return;
    }

    // ... existing listeners ...

    // NEW: Register update listener for cursor correction
    const cursorCorrectionUnregister = editor.registerUpdateListener(
        ({ editorState, tags }) => {
            // Early exit if not in Regular mode (WYSIWYG)
            if (mode !== EditorModes.WYSIWYG) return;

            // Skip if this was triggered programmatically
            if (tags.has(EDITOR_TAGS_USED.programaticDoRunChanges)) return;

            // Skip if editor is empty
            if (editorState.isEmpty()) return;

            // Check and correct cursor if needed
            correctCursorIfNeeded(editor);
        },
        {
            tag: [EDITOR_TAGS_USED.programaticDoRunChanges],
            skipTransforms: true,
        },
    );

    const cleanup = () => {
        wysiPreview();
        unregisterTransformWhileTyping();
        maintainMetadata();
        debouncedMaintainMetadata();
        redirectParaInsertionToLineBreakUnregister();
        lints();
        keyDownUnregister();
        moveToAdjacentNodesWhenSeemsAppropriateUnregister();
        pasteCommand();
        lockImmutablesOnCut();
        syncRefScrollUnregister();
        cursorCorrectionUnregister(); // NEW - add this
    };

    return cleanup;
}, [mode, markersViewState, editor, markersMutableState, /* ... other deps */]);
```

Key requirements:
- Only run in WYSIWYG mode (mode === EditorModes.WYSIWYG)
- Skip programmatic updates using EDITOR_TAGS_USED.programaticDoRunChanges
- Skip empty editors using editorState.isEmpty()
- Use skipTransforms: true to avoid triggering on every transform
- Add cursorCorrectionUnregister to cleanup function
- Return cleanup function to unregister listener on unmount

Dependencies:
- Previous ticket (helper functions must be added first)
- src/app/data/editor.ts - EditorModes, EDITOR_TAGS_USED

Acceptance:

- Editor updates trigger correctCursorIfNeeded() when appropriate
- No performance issues (listener early exits efficiently)
