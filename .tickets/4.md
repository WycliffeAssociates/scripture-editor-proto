# [Epic] Refactor: Extract ProjectIndexer

Extract database indexing logic from ProjectImporter into a dedicated ProjectIndexer service to fix architectural violations and improve separation of concerns.

## Design

This epic addresses the architectural violation where ProjectImporter (in src/core) imports from src/app/db/api.ts. The refactor will move indexing logic to src/app/domain/project/ProjectIndexer.ts and clean up ProjectImporter to only handle file I/O.

## Acceptance Criteria

ProjectIndexer service created and used throughout the codebase. ProjectImporter no longer imports from src/app. All tests pass and manual verification successful.
